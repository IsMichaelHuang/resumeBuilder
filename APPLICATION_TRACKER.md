## Application Checklist Template

For each application, complete this checklist:

### Pre-Application (5-10 min per job)
- [ ] Read full job description
- [ ] Identify 5-10 key technical requirements
- [ ] Choose appropriate resume version:
  - AI/ML keywords (PyTorch, LLM, ML) → `ai-ml/main.pdf`
  - Full-stack keywords (React, Node, MERN) → `fullstack/main.pdf`
  - Backend/Systems keywords (APIs, Linux, Infrastructure) → `backend/main.pdf`
  - General SWE/New Grad → `general-swe/main.pdf`
- [ ] Check if keywords match resume (add to skills.tex if needed)
- [ ] Recompile if you made changes: `cd versions/[version] && pdflatex main.tex`

### Application
- [ ] Apply on **company website** (not just LinkedIn Easy Apply)
- [ ] Use descriptive filename: `Michael_Huang_Resume_[RoleType].pdf`
- [ ] Customize cover letter if required (templates in `/coverLetter/`)
- [ ] Save job description URL and requirements

### Post-Application
- [ ] Add to tracker above with status "Applied"
- [ ] Set follow-up reminder for 1 week later
- [ ] Connect with recruiter/hiring manager on LinkedIn (optional)

## Status Definitions

- **Applied**: Submitted application
- **Viewed**: Recruiter viewed your profile
- **Screening**: Phone screen scheduled
- **Interview**: Technical interview scheduled
- **Offer**: Offer received
- **Rejected**: Not moving forward
- **No Response**: No reply after 2 weeks

## Pro Tips

1. **Apply Monday-Wednesday, 6-10 AM PT** - First applications get reviewed
2. **Direct applications > LinkedIn Easy Apply** - 2-3x higher response rate
3. **Tailor > Volume** - 10 tailored apps > 50 generic ones
4. **Follow up after 1 week** - Polite email: "Checking on my application status"
5. **Track everything** - You'll forget what you applied to
6. **Learn from rejections** - If 0 responses after 20 apps, revisit strategy
